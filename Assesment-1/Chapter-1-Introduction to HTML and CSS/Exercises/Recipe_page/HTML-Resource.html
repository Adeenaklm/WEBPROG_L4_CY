HTML & CSS Quick Reference Guide

A concise guide to essential HTML tags and CSS properties.

HTML Tags

  • <a> - <a href="url" target="_blank">Link</a>
    Defines a hyperlink with a new tab option.
  • <div> - <div>Content</div>
    Defines a block container.
  • <img> - <img src="image.jpg" alt="Description">
    Embeds an image.
  • <meta> - <meta charset="UTF-8">
    Defines metadata about the HTML document, such as character encoding.
  • <p> - <p>This is a paragraph.</p>
    Defines a paragraph.
  • <h1>-<h6> - <h1>Heading</h1>
    Defines headings from largest to smallest.
  • <ul>, <ol>, <li> - <ul><li>Item</li></ul>
    Defines lists.
  • <footer> - <footer>Footer content</footer>
    Defines a page footer.
  • <link> - <link rel="stylesheet" href="styles.css">
    Links an external CSS file to an HTML document.
  • <section> - <section>Section content</section>
    Defines a section in a document.

CSS Properties

  • display - display: flex;
    Used for flexible layout structures.
  • grid-template-columns - grid-template-columns: 1fr 2fr;
    Defines a two-column grid layout.
  • font-size - font-size: 16px;
    Defines text size.
  • font-weight - font-weight: bold;
    Defines text thickness.
  • text-align - text-align: center;
    Aligns text.
  • color - color: red;
    Sets text color.
  • background-color - background-color: blue;
    Sets background color.
  • border - border: 1px solid black;
    Defines border.
  • padding & margin - padding: 10px; margin: 10px;
    Spacing around elements.
  • position - position: relative;
    Controls element positioning.
