HTML & CSS Quick Reference Guide

Learn about essential HTML tags and CSS properties

HTML Tags

  • <a> - <a href="url">Link</a>

    Defines a hyperlink.

  • <div> - <div>Content</div>

    Defines a section or division.

  • <img> - <img src="image.jpg" alt="Description">

    Embeds an image.

  • <h1> to <h6> - <h1>Heading</h1>

    Defines headings from 1 (largest) to 6 (smallest).

  • <ul> - <ul><li>Item</li></ul>

    Defines an unordered list.

  • <ol> - <ol><li>Item</li></ol>

    Defines an ordered list.

  • <p> - <p>Text</p>

    Defines a paragraph of text.

  • <form> - <form><input></form>

    Defines a form for user input.

  • <span> - <span>Text</span>

    Defines a section of inline text.

  • <table> - <table><tr><td>Data</td></tr></table>

    Defines a table structure.

CSS Properties

  • color - color: red;

    Sets the color of the text.

  • background-color - background-color: #fafafa;

    Sets the background color of an element.

  • font-size - font-size: 18px;

    Defines the font size of text.

  • margin - margin: 20px;

    Sets the margin around an element.

  • padding - padding: 15px;

    Sets padding inside an element.

  • border - border: 1px solid black;

    Sets a border around an element.

  • text-align - text-align: center;

    Aligns text within an element.

  • display - display: block;

    Defines the display style of an element (e.g., block, inline).
